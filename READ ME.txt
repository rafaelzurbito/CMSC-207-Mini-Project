Personal Web Profile – Rafael A. Zurbito

Project Description

A personal web profile created using HTML, CSS, and JavaScript.
It introduces Rafael A. Zurbito, his professional background, hobbies, and contact information.

The website demonstrates the use of basic web development concepts such as:

HTML page structure
Internal page navigation using anchor links
Image display and interaction
Simple JavaScript functionality
External CSS styling

The webpage serves as a basic personal portfolio or profile page that highlights professional experience and personal interests.

Features

Navigation Menu

allows users to quickly jump to different sections of the webpage:

About Me
Hobbies & Interests
Contact Information

This is implemented using anchor links that connect to section IDs.

Interactive Profile Image

The profile picture has a simple JavaScript image swap function.

When the user clicks the image:

The image changes to a different picture.
Clicking again returns the image to the original photo.

This is done using the onclick event and a JavaScript function.
